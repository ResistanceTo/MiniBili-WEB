# 隐私政策

**最后更新日期：** 2025年10月12日

## 1. 简介

MiniBili 是一个免费、无广告的 Apple 生态第三方哔哩哔哩客户端。我们深知隐私的重要性，并承诺高度保护您的个人隐私与数据安全。

## 2. 数据收集声明

**我们不收集、不存储、也不传输任何用户的个人数据。**

具体说明如下：

- ❌ **不收集个人信息**：我们不收集您的姓名、邮箱、手机号或任何身份标识信息。
- ❌ **不收集使用数据**：我们不追踪您的浏览历史、观看记录或应用使用习惯。
- ❌ **不集成追踪 SDK**：应用内没有集成任何第三方统计、分析或广告追踪组件。
- ❌ **无第三方中间服务器**：我们没有自建的后端中间服务器，所有网络通信均直接发送至哔哩哔哩（Bilibili）官方 API。

## 3. 本地存储

应用仅在您的设备本地保存运行所需的必要配置。这些数据永远停留在您的设备中，绝不会上传至任何第三方服务器：

- **账号登录凭证**（仅用于保持哔哩哔哩登录状态，通过 Apple Keychain 加密存储）
- **应用偏好设置**（如播放器设置、主题外观等）
- **本地缓存数据**（如临时图片与视频缓存）

您随时可以通过从设备中卸载本应用来完全清除上述所有本地数据。

## 4. 第三方 API 服务

MiniBili 使用哔哩哔哩（Bilibili）官方 API 获取视频与动态数据。当您使用本应用时：

- 您的网络请求直接建立在您与哔哩哔哩官方服务器之间；
- 相关通信遵守哔哩哔哩官方的隐私政策与服务条款。
